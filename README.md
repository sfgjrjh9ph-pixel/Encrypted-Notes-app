# Encrypted-Notes-app
Encrypted Notes is a secure note-taking application that allows users to store personal information, ideas, passwords, journal entries, and other sensitive data in an encrypted vault. Protected by a master password, all notes are encrypted before being saved to disk, ensuring that only the user can access their contents. 

The application provides an easy way to create, edit, search, organize, and manage notes while maintaining privacy and security, making it a practical tool for safely storing confidential information in everyday life.
